# Basit Next.js Blog — Gürhan Savaş Akdağ

Bu depo basit ve modern bir Next.js blog örneğidir. `Gürhan Savaş Akdağ` isimli 13 yaşındaki yazar için örnek içerik içerir.

Çalıştırma (Windows PowerShell):

```powershell
cd c:/Users/gorke/Desktop/projeler
npm install
npm run dev
```

Ardından tarayıcıda `http://localhost:3000` adresini açın.

Notlar:
- Bu proje üçüncü parti bağımlılıkları minimal tutar — yalnızca `next`, `react`, `react-dom` kullanılır.
- İçerikler `data/posts.js` içinde yer almaktadır. Yeni yazılar eklemek için bu dosyayı güncelleyin.
